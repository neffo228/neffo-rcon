# nefforcon
```about
NeffoRcon - это инструмент для подключения к RCON'у майнкрафт серверов
```

Если у вас вылетает nefforcon, и не работает то попробуйте это:
```$ pip install mcrcon```
После установки перезапустите NeffoRcon, и должно работать


Пароль: ```neffo```
